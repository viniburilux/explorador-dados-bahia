Metodologia e dados

Como os dados foram coletados

Pipeline automatizado que consulta a API pública do PNCP. Filtra contratos por modalidade Inexigibilidade (cód. 9) — a forma legal de contratar artistas sem licitação. Dados coletados mês a mês para evitar timeouts.

Como os dados foram classificados

Cada contrato recebe uma categoria automática baseada no campo objetoCompra: artístico, saúde, consultoria, locação, manutenção. Contratos de órgãos estaduais/federais e outliers de valor são marcados com flags — sem excluir dados.

Cobertura

2025: janeiro a dezembro — 21.797 contratos, 273 municípios.
2026: janeiro a abril — 6.509 contratos, 157 municípios.
Total: 28.306 contratos de inexigibilidade analisados.

Transparência e verificação

Todo contrato tem um número de controle PNCP verificável em pncp.gov.br. Os dados são públicos e de acesso livre. Este projeto não faz acusações — apenas organiza o que já é público.
